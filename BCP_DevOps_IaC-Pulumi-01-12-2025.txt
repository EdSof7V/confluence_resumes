# Guía de Manipulación de Datos con Python

## Idea Principal
Introducción práctica a la manipulación de datos con Python, explorando el uso de **NumPy**, **Pandas**, y la visualización de datos mediante **Matplotlib**.

## Componentes Principales

### NumPy
Permite realizar operaciones matemáticas avanzadas sobre arrays multidimensionales.

- Los arrays de NumPy requieren elementos del mismo tipo
- Las operaciones entre arrays se realizan elemento por elemento

### Pandas
Facilita el trabajo con datos tabulares mediante DataFrames y Series.

#### Operaciones Fundamentales
- Filtrado de DataFrames con múltiples condiciones
- Uso de `join` para combinar datos de distintas fuentes

#### Funciones Clave
- `value_counts()`: Muestra la frecuencia de valores únicos
- `describe()`: Genera estadísticas descriptivas básicas
- `drop()`: Elimina columnas o filas innecesarias
- `astype()`: Convierte tipos de datos entre columnas

### Matplotlib
Herramientas para crear gráficos y personalizarlos.

#### Tipos de Gráficos
- **Box plots**: Identifican outliers y muestran la distribución de datos
- **Scatterplots**: Visualizan relaciones entre variables cuantitativas
- **Histogramas**: Representan la distribución de frecuencias

#### Personalización
- Uso de colores, marcadores y anotaciones con `plt.text()`
